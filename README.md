# Digital Chameleon

This repository contains the **Deepfake Deflector** project, a real-time adversarial
protection system designed for live webcam feeds. The full application lives in the
`deepfake-deflector/` subdirectory; read `deepfake-deflector/README.md` for details
on installation, usage, and demos.

### Contents

- `deepfake-deflector/` – main application source code, tests, and assets
- `.gitignore` – patterns to exclude from version control
- `LICENSE` – MIT license for the project

Feel free to clone the repo and explore the `deepfake-deflector` folder to run the
project or inspect the code.  

**Project URL:** https://github.com/Mahesh897620/Digital-Chameleon
